# fashion-ecommerce-profit-analysis
# ファッションEコマース商品の収益最大化分析

## 概要

本プロジェクトは、特定のファッションEコマースプラットフォーム（例: ASOS）の商品データを利用し、収益を最大化するためのデータ駆動型分析と具体的な戦略提案を行うものです。KGI（重要目標達成指標）を「収益の最大化」とし、データクリーニングから探索的データ分析、そして現場で実行可能なレベルのビジネス提案までの一連のプロセスをGoogle Colab上で実施しました。

## 目的

1.  **データ品質の確認と向上:** 提供された生データ（`products_asos.csv`）を読み込み、欠損値処理やデータ型変換などのデータクリーニングを実施し、分析に適した形式に整形する。
2.  **現状把握と課題特定:** 商品カテゴリ別の商品数や価格帯の分布を分析し、収益機会や潜在的な課題を特定する。
3.  **主要カテゴリの再定義:** 粒度が細かすぎるカテゴリ情報を、よりビジネス視点での分析に適した「主要カテゴリ」に再分類し、商品ポートフォリオをより明確に把握する。
4.  **収益改善提案:** 分析結果に基づき、高価格帯商品の戦略強化、低価格帯商品の販売数最大化、未分類商品の機会創出など、具体的なアクションプランと予測されるビジネスインパクトを提案する。

## 使用データ

-   `products_asos.csv`: ファッションEコマースプラットフォーム（ASOSと想定）の商品データ。以下の情報が含まれます。
    -   `url`: 商品ページのURL
    -   `name`: 商品名
    -   `size`: サイズ情報
    -   `category`: 商品カテゴリ（元データでは粒度が細かい）
    -   `price`: 価格
    -   `color`: 色
    -   `sku`: SKU (Stock Keeping Unit)
    -   `description`: 商品説明
    -   `images`: 商品画像URL

## 分析プロセス

### 1. データ読み込みと初期確認

アップロードされた`archive (1).zip`ファイルを解凍し、`products_asos.csv`をPandas DataFrameとして読み込みました。初期の`df.info()`および`df.describe()`により、データ構造、欠損値の有無、および`price`列が`object`型であるという問題点が判明しました。

### 2. データクリーニング

**「現場の泥臭いアクション」:**

-   **欠損値処理:** 全てのカラムで18件の欠損値が見つかったため、分析の精度を確保するため、該当する行を削除しました。
-   **`price`列の型変換:** `price`列が`object`型であったため、収益分析のためには数値型への変換が必須でした。カンマを除去し、`pd.to_numeric(errors='coerce')`を使用して数値に変換しました。この過程で、数値に変換できない値（非数値データ）を含む12449件のデータが特定され、これらも削除されました。これにより、残りのデータは`float64`型として価格情報を正確に表現できるようになりました。

    *この変換により大幅にデータ数が減少しましたが、これは`price`列のデータ品質が低いことを示しており、**データソース側でのデータ入力規則の厳格化**、または**データ取得時の前処理強化**が必要であることを示唆します。実務では、この段階でデータ提供元との連携を取り、データの整合性確保に努めます。*

### 3. カテゴリ分析 (初期)

クリーニング済みのデータ`df_cleaned`を使用し、以下の分析を行いました。

-   **カテゴリ別の商品数:** 元の`category`列における商品数の分布を調査。
-   **カテゴリ別の平均価格:** 元の`category`列における平均価格の分布を調査。

**初期分析からの課題:**

元の`category`列は、個別の商品名やブランド名に近い粒度で情報が含まれており、上位概念での分析が困難であることが判明しました。これは、本来「Jacket」や「Dress」といった大まかなカテゴリとして認識されるべきデータが、詳細な商品情報と混在していることを示しており、データ構造の整合性に課題がありました。

### 4. 主要カテゴリの再定義と分析

**「データ構造の整合性」を疑う視点と具体的なアクション:**

-   `category`列と`name`列を結合したテキストデータから、正規表現とキーワードマッチングを用いて、より汎用的な`main_category`（例: 'Dress', 'Jacket', 'Top', 'Socks'など）を抽出する関数`get_main_category`を実装しました。
-   これにより、粒度が細かすぎたカテゴリをビジネス分析に適したレベルに集約し、以下の分析を行いました。
    -   **主要カテゴリ別の商品数トップ15:** どの主要カテゴリに多くの商品が存在するかを可視化。
    -   **主要カテゴリ別の平均価格トップ10/ワースト10:** 主要カテゴリごとの平均価格を比較し、高価格帯と低価格帯の傾向を把握。

**「現場の泥臭いアクション」:**

このようなカテゴリ抽出は、しばしば手作業でのキーワードリスト作成や、正規表現の試行錯誤が必要です。完璧な分類は困難な場合もありますが、ビジネス上の意味を持つ程度に集約することが重要です。

## 分析結果と提案

### 課題定義

1.  **「Other」カテゴリの存在:** 主要カテゴリに分類しきれなかった`Other`カテゴリが商品数で上位に位置しており、潜在的な機会を見逃している可能性があります。
2.  **カテゴリ間の価格差と商品数のバランス:** `Dress`のように商品数が多いが平均価格が中程度のカテゴリ、`Biker Jacket`のように平均価格が高いが商品数が少ないカテゴリ、`Socks`のように商品数が少ないかつ平均価格も低いカテゴリが存在し、これらのバランス最適化が必要です。

### 仮説

1.  **仮説3（「Other」カテゴリの深掘り）：** `Other`カテゴリに含まれる商品には、新しいトレンドやニッチな需要を捉える機会がある。
2.  **仮説4（製品タイプ別のポートフォリオ最適化）：** 各主要カテゴリの特性に応じた戦略的なリソース配分とマーケティングで、全体の収益性を向上できる。

### 具体的な提案とアクションプラン

#### 提案1：高価格帯商品の戦略強化

-   **対象カテゴリ:** `Biker Jacket`, `Puffer Jacket`, `Jacket`, `Trench Coat`, `Coat` など（平均価格が高いカテゴリ）
-   **具体的なアクション:**
    -   **販売チャネルの最適化:** VIP顧客向けプロモーション、専門販売員の配置。
    -   **コンテンツマーケティング強化:** 商品のストーリー（素材、デザイン、背景）を伝える高解像度コンテンツ、動画、専門家レビューの作成。
    -   **限定性・希少性の演出:** 期間限定・数量限定販売、コラボレーション商品の導入。
    -   **ターゲット広告:** 高所得者層やファッション感度の高い層に絞ったSNS広告、ファッション誌広告。
-   **収益インパクトと投資対効果:** 高価格帯商品のコンバージョン率1%向上で年間売上〇〇円増加。コンテンツ作成・広告費用〇〇円に対し、十分なROIを期待。

#### 提案2：低価格帯商品の販売数最大化とコスト最適化

-   **対象カテゴリ:** `Socks`, `Brief`, `Activewear` など（平均価格が低いカテゴリ）
-   **具体的なアクション:**
    -   **セット販売・バンドル販売:** 複数商品のセット割引、関連商品との抱き合わせ販売。
    -   **クロスセル・アップセル促進:** 低価格商品購入者への関連中価格帯商品レコメンド強化。
    -   **SNSでの拡散戦略:** インフルエンサーマーケティング、UGC活用キャンペーン。
    -   **物流コストの最適化:** 効率的な在庫管理、梱包材見直し、配送ルート最適化、サプライヤー交渉による仕入れコスト削減。
-   **収益インパクトと投資対効果:** 低価格帯商品の平均購入点数0.5点増加、販売数5%向上で年間売上〇〇円増加。仕組み導入・キャンペーン費用〇〇円に対し、高いROIを見込む。

#### 提案3：未分類商品（「Other」カテゴリ）の機会創出と分類精度の向上

-   **対象カテゴリ:** `Other`カテゴリに含まれる商品
-   **具体的なアクション:**
    -   **詳細レビュー:** `Other`カテゴリ内の商品名、説明、URLを手動でレビューし、共通の特徴や潜在的なカテゴリを発見。
    -   **キーワードの追加・ロジックの改善:** レビュー結果に基づき、`get_main_category`関数に新たなキーワードを追加し、分類ロジックを改善。
    -   **新しいカテゴリの創設:** 特定のニッチな商品群が繰り返し見つかる場合、新たな主要カテゴリとして定義し、専用の戦略を検討。
-   **収益インパクトと投資対効果:** `Other`カテゴリの1754件の商品が適切に分類され、そのうち10%が新たなカテゴリ戦略により売上が5%向上した場合、年間売上は〇〇円増加する可能性がある。分類作業にかかる工数（人件費）は〇〇円と見積もられ、新たな売上機会の創出を考慮すると十分な投資対効果が見込めます。

#### 提案4：主要カテゴリ別のポートフォリオ最適化と個別戦略

-   **対象カテゴリ:** 全ての主要カテゴリ
-   **具体的なアクション:**
    -   **高価格帯アウターウェア:** 品質、デザイン、ブランドストーリーを強調するプレミアムコンテンツ、限定性・希少性の演出、高所得層向けターゲット広告、店舗スタッフ研修による深い知識提供。
    -   **高ボリューム中価格帯商品（Dress, Top, T-Shirtなど）:** 定期プロモーション、関連商品とのバンドル販売・コーディネート提案、SNSでのUGC活用、効率的な在庫管理と売れ筋商品補充、返品率分析による品質改善。
    -   **低価格帯商品（Socks, Brief, Activewearなど）:** まとめ買い割引、サブスクリプションモデル検討、高単価商品との組み合わせインセンティブ、利便性・コストパフォーマンスを訴求する広告、物流コスト最適化。
-   **収益インパクトと投資対効果:** これらの施策により、高価格帯商品の売上高が10%増加、中価格帯商品の平均客単価が5%向上、低価格帯商品の販売数が15%増加した場合、年間で〇〇円の売上増加が見込まれる。各戦略にかかる費用は合計で〇〇円と見積もられ、ビジネス成長への貢献は極めて大きいと判断できます。

## 今後の展望

-   **予測モデルの構築:** 売上予測モデルや需要予測モデルを構築し、在庫最適化やマーケティング予算配分の精度を高めます。
-   **A/Bテストの実施:** 提案された施策についてA/Bテストを実施し、効果を定量的に検証・改善します。
-   **詳細な収益性分析:** 各カテゴリ、各商品の粗利率などのデータがあれば、より精緻な収益性分析とコスト最適化の提案が可能となります。
